# USD Currency Converter - Download

Download the USD Currency Converter skill for Claude.

## Current Release

**Download:** [usd-currency-convert.zip](usd-currency-convert.zip)

**Supported Currencies:**
- **EUR** (Euro) - 2010-present
- **PLN** (Polish Zloty) - 2012-present
- **AUD** (Australian Dollar) - 2010-present

**Coverage:** 11,500+ daily exchange rates

---

## Installation

### Option 1: Claude Desktop (Recommended)

1. Download [usd-currency-convert.zip](usd-currency-convert.zip)
2. Upload the ZIP file directly through Claude Desktop
   - The skill will be automatically extracted and installed

### Option 2: Manual Installation

1. Download [usd-currency-convert.zip](usd-currency-convert.zip)
2. Extract the contents
3. Copy the extracted folder to your Claude skills directory:
   - **Windows:** `%USERPROFILE%\.claude\skills\usd-currency-convert`
   - **macOS/Linux:** `~/.claude/skills/usd-currency-convert`

### Option 3: Claude Web/API

1. Download [usd-currency-convert.zip](usd-currency-convert.zip)
2. Upload through the Claude web interface
3. The skill will be automatically available

## Usage

Once installed, ask Claude to convert currencies:

```
"Convert $250 to EUR on March 20, 2024"
"What's the exchange rate for EUR on March 20, 2024?"
"Convert $1500 to Polish z≈Çoty on January 15, 2025"
"Convert $1000 USD to AUD as of December 1, 2024"
```

Claude will automatically use this skill to provide accurate conversions based on official central bank rates.
